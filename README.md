# Matatu Rush: Nairobi Streets

A simple 2D endless driving game concept — weave a matatu through Nairobi traffic, dodge police roadblocks, pick up passengers for fare, and spend earnings on upgrades in the Garage.

**▶ Play now: https://abdisalam-m.github.io/matatu-rush-nairobi-streets/**

## Play

Open `index.html` in a browser, or play directly at the link above (served via GitHub Pages from `main` / root).

## Features

- Left/right lane movement (arrow keys, A/D, on-screen buttons, or swipe)
- Traffic cars to dodge
- Police roadblocks (bigger penalty than traffic)
- Passengers as collectibles (fare + score)
- Scoring system with a persisted best score
- Upgrade shop ("The Garage"): Engine Tune, Reinforced Body, Extra Seats, Sacco Horn -- bought with Ksh earned per run, saved in `localStorage`
